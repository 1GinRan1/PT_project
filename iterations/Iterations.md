# Roadmap

## Iteration 1: Concept & Game Design
* **Goals:** Formulate a clear vision of the game, describe the rules, core mechanics, and visual style.
* **Result:** Approved game concept.
* **Risks:** * Unfeasible project scope for a small team.
  * Discrepancies in the vision of the final product among team members.

## Iteration 2: Technical Architecture
* **Goals:** Lay the code foundation in Unity and create rough level layouts.
* **Result:** Basic Unity project, configured class architecture (for the character, enemies, and soul system).
* **Risks:** * Issues with setting up version control (Git) for collaborative work in Unity.

## Iteration 3: Core Mechanics 
* **Goals:** Implement core mechanics, movement physics, and the combat system.
* **Result:** The first working prototype where the character can perform basic movements and interact with test objects/enemies.
* **Risks:** * Presence of input lag.
  * Collision bugs (e.g., falling through textures).
