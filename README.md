# 2D platformer 

## 🎮 Overview
This project is a 2D platformer developed for PC using the **Unity Engine**. It is a direct evolution and refinement of a bachelor's thesis project, focusing on polished gameplay mechanics, responsive controls, and immersive level design.

The game challenges players to navigate through complex environments, utilizing various movement abilities to overcome obstacles and progress through levels.

## 🚀 Key Features
* **Precision Platforming:** Fine-tuned character controller for tight and responsive movement on PC.
* **Legacy-Driven Design:** Built upon a solid foundation of academic research and initial Unity prototyping.
* **Dynamic Environments:** Interactive objects and hazards that require strategic timing.
* **Optimized for Desktop:** Tailored specifically for PC hardware with support for keyboard/mouse and controller inputs.

## 🛠 Tech Stack
* **Engine:** Unity 2022.x 
* **Language:** C#
* **Platform:** Windows / PC
* **Version Control:** Git

## 📖 Development Background
The core of this project originated as a bachelor's degree work. This version represents a significant upgrade in terms of code architecture, asset quality, and overall gameplay depth, moving from an academic prototype to a more complete software product.
