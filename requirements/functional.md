# Functional Requirements

* **Level Availability:** The game must contain playable levels for the player to complete.
* **Character Control:** The game must provide full user control over the movement of the game character.
* **Progression System:** The game must provide player progression (e.g., finding items) throughout the levels.
* **Combat System:** The game must include a combat system and enemies to ensure an engaging player experience.
* **Save System:** The game must save the player's global progress during the walkthrough.
