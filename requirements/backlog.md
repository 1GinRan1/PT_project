# User Stories

* **Story #1:** As a player, I want each of my runs to be unique so that the game does not become monotonous due to repeating the same routes.
* **Story #2:** As a player, I want to feel a progression of difficulty with each new level to provide a challenge to my skills.
* **Story #3:** As a PC user, I want to be able to customize the game settings to make the gameplay comfortable for me.
